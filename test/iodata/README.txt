This folder holds sample files for input/output testing.

* libjpeg
  testimg.jpg is from the libjpeg library archive, release 6b,
  available online at the following address:
      http://www.ijg.org/files/jpegsrc.v6b.tar.gz
  testimg_gray.jpg is a grayscale version of this image.

* libtiff
  These files are from the libtiff library archive test images,
  available online at the following address:
      ftp://ftp.remotesensing.org/pub/libtiff/pics-3.8.0.tar.gz
