This modules performs a singular (level-lines based) interpolation of
an image. The $input$ image is interpolated in the regions specified
in the $holes$ mask image. The \verb+drawocclusion+ module can
be used to create such a mask image of the holes. See [1] for 
a detailed description of the algorithm.

\medskip

[1] S. Masnou, {\it Filtrage et désocclusion d'images par méthodes d'ensembles
de niveau}, PhD dissertation, Ceremade, Université Paris-Dauphine, 1998.
